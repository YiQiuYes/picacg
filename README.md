# PicACG

一款基于Flutter框架开发的高性能、美观、用户友好的漫画阅读应用。本项目旨在提供流畅的漫画浏览体验，同时保持应用界面简约现代。

## 📥 安装

```bash
# 克隆仓库
git clone https://github.com/YiQiuYes/picacg.git picacg

# 切换到项目目录
cd picacg

# 安装依赖
flutter pub get

# 运行应用
flutter run
```
